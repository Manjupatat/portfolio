# portfolio

# Creative Portfolio Website

A modern, animated portfolio website built with HTML, CSS (Bootstrap), and Vanilla JavaScript.

## 📁 File Structure
```
portfolio/
│
├── index.html          # Main HTML file
├── style.css           # All CSS styles
├── script.js           # All JavaScript functionality
└── README.md           # This file
```

## 🚀 Setup Instructions

1. **Create a folder** for your portfolio project
2. **Save the files**:
   - Save the HTML code as `index.html`
   - Save the CSS code as `style.css`
   - Save the JavaScript code as `script.js`
3. **Make sure all files are in the same folder**
4. **Open** `index.html` in your web browser

## ✨ Features

### Visual Effects
- Animated cursor glow effect
- Floating background orbs with parallax
- Glassmorphism design with backdrop blur
- Gradient animations throughout

### Interactive Elements
- Smooth scroll navigation
- Active link highlighting
- Typing effect on hero subtitle
- Animated statistics counter
- Ripple effect on button clicks
- 3D tilt effect on project cards
- Scroll-triggered animations

### Sections
1. **Hero** - Introduction with animated stats
2. **Projects** - Featured projects showcase
3. **Skills** - Technology expertise
4. **Contact** - Social media links

## 🎨 Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --secondary-gradient: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
    --dark-bg: #0a0a0a;
}
```

### Content
Edit text directly in `index.html`:
- Personal information in hero section
- Project details in projects section
- Skills in skills section
- Contact links in contact section

## 📦 Dependencies

All dependencies are loaded via CDN:
- Bootstrap 5.3.2
- Font Awesome 6.4.2

## 🌐 Deployment

### GitHub Pages
1. Create a GitHub repository
2. Upload all files
3. Go to Settings → Pages
4. Select main branch
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Netlify
1. Drag and drop your folder to Netlify
2. Your site will be live instantly

### Vercel
1. Import your GitHub repository
2. Deploy with one click
